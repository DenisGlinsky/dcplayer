# ADR

Architecture Decision Records используются для решений, которые меняют:

- trust boundary;
- формат объектов и их инварианты;
- выбор библиотек;
- политики безопасности;
- аудио/видео pipeline;
- стратегию watermark и detector.

Формат ADR:
1. Контекст
2. Решение
3. Последствия
4. Альтернативы
5. Статус
