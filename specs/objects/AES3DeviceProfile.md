# AES3DeviceProfile

## Назначение

Профиль устройства/адаптера для 24-bit AES3 output.

## Канонические поля

- `device_id` — строка
- `channel_count` — целое > 0
- `sample_rate_hz` — целое > 0
- `bit_depth` — целое > 0
- `framing_mode` — строка

## Инварианты

- Bit depth и sample rate должны быть поддержаны устройством.
- Профиль не описывает частные драйверные секреты.

## Связи с другими объектами

- Используется T07c.

## Каноническая сериализация

- JSON snake_case.

## Причины отклонения / ошибки

- отсутствует обязательное поле;
- нарушен инвариант;
- формат времени, enum или идентификатора невалиден;
- объект противоречит связанным контрактам;
- сериализация недетерминирована.
