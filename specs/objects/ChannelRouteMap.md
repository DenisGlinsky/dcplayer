# ChannelRouteMap

## Назначение

Label-driven маршрутизация PCM каналов на внутренние и внешние линии.

## Канонические поля

- `profile_id` — строка
- `input_labels` — массив строк
- `output_routes` — массив mapping entries
- `sample_rate_hz` — целое > 0
- `bit_depth` — целое > 0

## Инварианты

- Маршруты не должны конфликтовать без явной fan-out policy.
- Label-driven routing обязателен; позиционное угадывание запрещено.

## Связи с другими объектами

- Используется PCM engine и AES3.

## Каноническая сериализация

- JSON snake_case.

## Причины отклонения / ошибки

- отсутствует обязательное поле;
- нарушен инвариант;
- формат времени, enum или идентификатора невалиден;
- объект противоречит связанным контрактам;
- сериализация недетерминирована.
