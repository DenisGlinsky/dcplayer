# SubtitleCue

## Назначение

Унифицированная модель cue для Interop и SMPTE Text.

## Канонические поля

- `cue_id` — UUID строка
- `start_time` — таймкод/таймстамп
- `end_time` — таймкод/таймстамп
- `payload_ref` — текст или bitmap ref
- `placement` — объект

## Инварианты

- end_time >= start_time.
- Cue не должен иметь пустой payload.

## Связи с другими объектами

- Используется Interop Subtitles и SMPTE Text.

## Каноническая сериализация

- JSON snake_case.

## Причины отклонения / ошибки

- отсутствует обязательное поле;
- нарушен инвариант;
- формат времени, enum или идентификатора невалиден;
- объект противоречит связанным контрактам;
- сериализация недетерминирована.
